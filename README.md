# Laminae (Archived)

> AI personality, safety, and red-teaming SDK written in Rust.

## What This Was

Laminae was a Rust SDK for defining AI agent personalities, enforcing safety constraints, and running red-team evaluations. It provided composable building blocks for teams building production AI systems that needed behavioral guardrails and adversarial testing.

## Stack

- Rust
- Cargo (build/test)

## Usage

This project is archived and no longer maintained.
